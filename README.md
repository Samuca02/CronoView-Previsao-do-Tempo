# [🌤️ CronoView - Previsão do Tempo](https://loquacious-lily-dd33b6.netlify.app)

## 🔗 Acesse o projeto
--------------------------------------------------------------------------------------------   
O sistema exibe informações climáticas em tempo real, como:

• Temperatura

• Condição do clima

• Umidade do ar

• Ícone dinâmico da previsão

-------------------------------------------------------------------------------------------
🛠️ Tecnologias utilizadas:

• HTML5

• CSS3

• JavaScript

• OpenWeatherMap API
